# vote_frontend
(仮)Voteフロントエンド

## [WIP]開発環境準備
1. nodeをインストール
2. このリポジトリを`$ git clone`
3. npm ci

## ブランチルールと開発フロー
1. developから開発用ブランチを切る
2. develop ← 開発ブランチでPRを出す
3. マージする
4. いい感じだったらmaster ← developでPRを出す
5. マージする
